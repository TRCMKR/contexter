# Contexter

Телеграм бот на базе LLM и LM Studio

## Реализованы
- Интеграция с LM Studio
- Поддержка контекста независимо для каждого пользователя
- Очистка контекста

## Команды
- `/start` - Начать работу с ботом
- `/model` - Вывести название текущей модели
- `/clear` - Очистить контекст
- Любое другое сообщение будет отправлено в LLM с учётом контекста

## Запуск
- Склонируйте репозиторий
- Установите зависимости:
```
pip install -r requirements.txt
```
- Создайте бота через `@BotFather` и получите токен
- Создайте файл `.env` по образу `.env.example` и заполните его
- Запустите LM Studio, загрузите нужную модель и запусти её локально
